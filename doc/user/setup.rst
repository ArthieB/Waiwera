***********************
Setting up a simulation
***********************

This section describes how to set up the various aspects of a Waiwera simulation.

All input data for a simulation are contained within one object in the Waiwera input JSON file. The various aspects of the simulation are specified by different named values within that object. These values are of various types, and many are objects themselves.
