************
Input schema
************

Below is the input `schema <http://json-schema.org/>`_ for validating Waiwera JSON input files. This serves as both a technical specification for the Waiwera JSON input file structure, and a means of validating input files automatically using JSON validation `tools <http://json-schema.org/implementations.html>`_ (e.g. libraries, text editor plugins or online validators).

This input schema is included as a JSON file (`input_schema.json`) in the `utils/` directory of the Waiwera source code.
