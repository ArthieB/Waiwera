*************
Time stepping
*************

.. specify start time, stop time (number or null)
.. time stepping: can specify time step sizes (constant value, or array)
.. or use adaptive stepping: set method (backward Euler, BDF2 etc.)
.. other time stepping parameters, including adaptor method
.. solvers: non-linear and linear solvers
.. non-linear: set min/max iterations, tolerance
.. linear: type, options, tolerance, max iterations
.. preconditioner: type, sub-preconditioner
